This project presents an intelligent and automated system for detecting and classifying cotton leaf diseases using a hybrid deep learning and ensemble learning framework. Cotton is one of the most important agricultural crops, especially in countries like India, where crop diseases significantly impact yield, quality, and farmer income. Early and accurate identification of cotton leaf diseases is crucial for timely intervention, but traditional manual inspection methods are slow, subjective, and impractical for large-scale farming. This project aims to address these challenges using modern artificial intelligence techniques.

The proposed system classifies cotton leaf images into four categories: Healthy, Bacterial Blight, Cotton Leaf Curl Virus, and Fusarium Wilt. The dataset used in this project is obtained from Kaggle and consists of real-world cotton leaf images. Extensive preprocessing is applied, including image resizing, normalization, and data augmentation, to ensure consistent input dimensions and improve model generalization. All images are resized to 224×224 pixels to match the input requirements of deep learning models.

The core of the system is a hybrid feature extraction approach using three powerful transfer learning models: VGG16, ResNet50, and DenseNet121, pre-trained on ImageNet. Instead of using these models for end-to-end classification, their fully connected layers are removed, and Global Average Pooling is applied to extract high-level discriminative features from each model. The extracted feature vectors are concatenated to form a comprehensive representation that captures diverse spatial and hierarchical information from cotton leaf images.

For classification, a soft voting ensemble classifier is employed, combining Logistic Regression, Support Vector Machine (SVM), and Random Forest algorithms. This ensemble strategy improves robustness by leveraging the strengths of multiple classifiers and reduces overfitting compared to single-model approaches. The final prediction is based on averaged class probabilities.

Experimental results demonstrate excellent performance, achieving an overall accuracy of approximately 97–99%, along with high precision, recall, F1-score, and ROC-AUC values across all classes. The proposed system provides a reliable, scalable, and accurate solution for cotton leaf disease detection and has strong potential for deployment in real-time agricultural decision-support systems, mobile applications, and precision farming platforms.
